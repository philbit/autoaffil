autoaffil provides the author/affiliation layout style common in physics and related fields: all authors appear in a single block with superscript numbers linking each name to a list of affiliations printed below — the style native to revtex4-2 with the superscriptaddress option, and familiar from journals such as Physical Review Letters. The package brings this style to the standard article class and compatible classes, without requiring a specialised document class.

Authors and affiliations are declared individually in the preamble using \addauthor and \addaffil. Identical affiliation strings are automatically deduplicated: each unique affiliation receives a number in order of first appearance, and authors sharing an affiliation automatically receive the same number. A single \maketitle call outputs the complete author/affiliation/notes block without any further effort.

Package options:

  • ranges — compress runs of three or more consecutive affiliation numbers to an en-dash range (e.g. 1–3); pairs and singles are unchanged
  • superaftercomma — place superscripts after the inter-author comma (revtex4-2 style) rather than before it
  • nobreak — wrap each name+superscript unit in \mbox{} to prevent mid-entry line breaks; inter-author spaces remain breakable
  • manual — suppress auto-insertion into \maketitle; place the author, affiliation, and note blocks manually via \printauthors, \printaffils, and \printnotes

The author font, inter-author separator, and inter-author spacing are all customisable via simple hook commands (\aafauthorfont, \aafauthorsep, \aafauthorspace).

Requirements: LaTeX2e and etoolbox (standard in TeX Live and MiKTeX).
